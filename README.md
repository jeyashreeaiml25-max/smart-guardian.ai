# Smart Guardian AI

Smart Guardian AI is a simple Python project that helps parents filter unsafe or harmful text content for children.

## Features

- Detects harmful words
- Filters inappropriate content
- Calculates safety score
- Age-based filtering system

## Technologies Used

Python
Text Processing
File Handling

## How to Run

1. Install Python
2. Run the program

python filter.py

## Project Purpose

This project demonstrates how basic content filtering can help protect children from unsafe digital content.
